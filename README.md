# AllDaysHolidays — MVP v3

W tej wersji poprawiono wygląd i dodano:
- sekcję „Gdzie lecieć?” z rekomendacjami kierunków,
- karty kierunków z ocenami: cena, pogoda, plaże, imprezy,
- ostrzeżenia „nie leć tam, jeśli…”,
- porównanie Malta vs Malaga vs Kreta,
- lepszy hero, nawigację i układ mobile,
- konto demo w prawym górnym rogu,
- paszport podróży: kraje, miasta, wyjazdy, odznaki,
- pływający chat AI,
- zapis demo w localStorage.

## Jak odpalić
Otwórz `index.html` w przeglądarce.

## Ważne
To jest nadal statyczny frontend. Prawdziwe logowanie, płatności, AI i oferty wymagają backendu oraz integracji API.
